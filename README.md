##작업하면서 어려웠던 점

일단 최대한 구글에 내가 원하는 효과를 검색해 봤고 적용해 봤을 때 왜 됐는지 이해를 하며 이유를 찾는데 꽤 많은 시간이 걸렸다.
아예 식을 찾아볼 수 없는 건 지인에게 도움을 청했다. 만날 수 있는 상황이 되지 못해서 미트로 만나서 수업처럼 진행했다.
이해하며 배우며 하다 보니 한 작동을 시키는 데에도 두세 시간 정도씩 걸렸던 것 같다.
