Resolutions

의도 했던 풀페이지 화면과 다르게 완성
arrow_right_alt 코드에 @media (min-width: 1600px)를 추가로 작성하여 작업.
수정이 필요한 부분에만 padding, width, height 값을 재지정해 줌.
pc 기준에서 화면을 줄일 때 vision 부분 이미지가 자연스럽게 이동하지 않고
이미지가 작아짐
arrow_right_alt position : absolute 값 기준을 right로 변경해 자연스럽게 이동하게 해 줌
width, height 값을 %에서 고정값인 px로 변경
swiper 이벤트값 적용 전 일렬로 배치 시 아이템이 밑으로 내려감
arrow_right_alt display:flex; 적용 후 아이템의 너비가 줄어들지 않도록 flex-shrink: 0; 적용
